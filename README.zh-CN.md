## 🎮 游戏玩法

### 📖 游戏概述

**Archaeological Explorer** 是一款基于历史考古的探索游戏。你将扮演一名考古学家，收集专业工具、规划探险路线，逐步发现世界各地的著名遗迹。

每次探险都是独特的——根据工具的探险权重选择合适的组合，你可能会发现吉萨金字塔、兵马俑、或是神秘的狄俄尼索斯剧场！

- **可发现遗迹**: 常见考古遗址，如吉萨金字塔、兵马俑

- **权重范围**: 15 - 50

- **难度**: ⭐⭐⭐

#### 🔴 大师探险

- **专家组合**: LiDAR + GPR + 磁力仪 + 其他 = 40+ 权重
- **难度**: ⭐⭐⭐⭐⭐

## 💡 游戏技巧

### 🎯 新手建议

1. **从基础开始**: 使用所有6个起始工具（8权重）进行初级路线
2. **携带限制重要**: 1级时只能携带3个工具！请明智选择
3. **权重策略**: 不要随意选3个工具——要计算总权重
   - 例：相机(1) + 铲子(2) + 桶(2) = 5 权重（仍然是初级）
   - 避免：3个随机重工具可能超过初级范围
4. **先升级**: 到达2-3级后再尝试高级路线
5. **阅读遗迹**: 每个遗迹都有真实的历史信息和引用！

### 🌟 进阶策略

1. **权重优化**：
   - 初级路线 (1-20): 混搭轻型工具，总权重≤20
   - 高级路线 (15-50): 添加泥刀、无人机等中型工具
   - 大师路线 (40+): 使用 LiDAR + GPR 等传奇工具

2. **携带槽位管理**：
   - 1-2级：只有3个槽位——优先选择高效工具
   - 3-4级：4-5个槽位——现在可以添加无人机或航空成像
   - 5级+：6-8个槽位——终于可以使用完整的传奇装备

3. **发现概率**：
   - 常见遗迹 (85-90%): 初级路线容易发现
   - 稀有遗迹 (70-80%): 需要高级路线，可能需要多次尝试
   - 隐藏遗迹 (20-25%): 仅限大师路线，非常具有挑战性！

4. **高效升级**：
   - 每次发现 = 25-50 EXP
   - 首次发现 = 额外奖励 EXP
   - 优先专注高概率遗迹以快速升级
